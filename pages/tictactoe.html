Tic | Tac | Toe

Tic Tac Toe game built in HTML, CSS, AngularJS and JavaScript. Game state is recorded via Firebase.

I used the ng-show and ng-hide AngularJS directives to show win notifications, which hides the "Tic Tac Toe" title at the top. The game is playable online, records win count and has a reset button that clears the Firebase records to start a fresh game.
