OfferUp!

Social borrowing platform, this app lets you add friends and search for items based on location or name. App built in Ruby on Rails, PostgreSQL, Elasticsearch/Searchkick, styled with HTML5/CSS3, SASS and Zurb Foundation. Mobile app built in AngularJS with the Ionic framework.

Alum.ly

Alumni social network for graduates of General Assembly. Features include Job board, Events board & social Forum. App built with Ruby on Rails, PostgreSQL, Amazon AWS/S3, Google Places Autocomplete, Google Maps & Javascript. Styled with HTML5/CSS3, including a custom grid system.

Personal Blog

A blog for personal musings, thoughts, and of course pictures of food! App featured a user and comment system. Built with Ruby on Rails, PostgreSQL, Amazon AWS/S3, stlyed with HTML5/CSS3 and the Twitter Bootstrap CSS framework. Deployed via Heroku.

Tic | Tac | Toe

X's and O's will never be the same again. Tic Tac Toe game built with HTML/CSS, JavaScript and AngularJS. Players are able to play each other online via Firebase database. Features include ng-show/hide for win notices, win counter and reset button.
